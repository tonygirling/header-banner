HTML in Customiser

The site relies on an HTML block in the Header when using the customiser.

In the Header part of the Customiser ensure there is an HTML widget on the bottom row.

Ensure the view is for Desktop.

Select the settings for the HTML widget and ensure the Text tab is in focus.

Enter the follwing markup.

<div id="themed-section-text-div-desktop"></div>

Now select Mobile and repeat the above but ensure the markup is as:-

<div id="themed-section-text-div-mobile"></div>
